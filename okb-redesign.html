Owlkids Books Redesign

Role:
UI/UX Design, Visual Design

Tools:
Sketch, Photoshop

Project:
I decided to take on the task of redesigning the current Owlkids Books website for a school project during my time in a User Interface design course. I felt that the current Owlkids Books site was outdated and could use a little facelift interms of design and usability. So I took on the task of re-thinking elements to make it more easier for the user to find what they are looking for as well as making the overall experience more pleaseing to the eye.

Simplified site map/user flow to make things easier for the user to find upon landing on the website.

Clean simple layout for the store items,
with easy category sub-navigation right up top.

Modal opens highlighting the chosen item
accompanied with additional informaition.
